Student Performance Analysis
Project Overview:
This project analyzes student performance using a dataset (`student_performance_data.csv`) with details such as:
- Student demographics (ID, Gender, Age)
- Study habits (Study Hours per Week, Attendance Rate)
- Academic results (GPA, Major)
- Lifestyle factors (Part-Time Job, Extra-Curricular Activities)

The goal is to explore relationships between study habits, attendance, and GPA, and to demonstrate how to manipulate data using pandas in Python.

Dataset:
The dataset contains 500 student records with the following columns:
StudentID
1 Gender
2 Age
3 StudyHoursPerWeek
4 AttendanceRate
5 GPA
6 Major
7 PartTimeJob
8 ExtraCurricularActivities



Installation:
Make sure you have Python 3 installed. Then install the required library:
pip install pandas
